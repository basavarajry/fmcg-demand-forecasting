How to run
Project folder:

c:\Users\basav\OneDrive\Desktop\fmcg-demand-forecasting

1. One-time setup
cd c:\Users\basav\OneDrive\Desktop\fmcg-demand-forecasting
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env


2. Generate sample data
python scripts/download_data.py --dataset synthetic


3. Train models (pick what you want)
Quick (baselines + ML + TFT):

python scripts/train.py --phases baseline ml tft
Everything (slower):

python scripts/train.py --phases baseline ml deep tft
Reports go to reports\model_comparison.csv.

4. Run the API
uvicorn src.api.main:app --reload --host 0.0.0.0 --port 8000
Swagger UI: http://localhost:8000/docs
Health: http://localhost:8000/health
Example forecast (default token is dev-token):

curl -X POST http://localhost:8000/forecast `
  -H "Authorization: Bearer dev-token" `
  -H "Content-Type: application/json" `
  -d "{\"history\":[{\"date\":\"2024-01-01\",\"sku_id\":\"SKU_0001\",\"dc_id\":\"DC_00\",\"demand\":100},{\"date\":\"2024-01-02\",\"sku_id\":\"SKU_0001\",\"dc_id\":\"DC_00\",\"demand\":110}],\"horizon\":7}"
5. Run the dashboard
In a second terminal (with venv active):

streamlit run dashboards/app.py
Open: http://localhost:8501

6. Run with Docker (optional)
docker compose up -d
Service	URL
API
http://localhost:8000
Dashboard
http://localhost:8501
MLflow
http://localhost:5000
7. Run tests
pytest tests/ -v
Makefile shortcuts (if you use make)
make install
make data
make train
make api
make dashboard
Notes
TFT / deep learning needs PyTorch and a GPU for reasonable speed. If install fails, start with:
python scripts/train.py --phases baseline ml
First run can take several minutes (especially TFT on CPU).
Activate the venv in every new terminal: .\.venv\Scripts\activate
